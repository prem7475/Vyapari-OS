# Vyapari OS

AI-powered Business Operating System for India-first entrepreneurs.

This repository is a monorepo containing:

- `apps/mobile` (Flutter)
- `apps/web` (Next.js marketing site)
- `apps/admin` (Next.js admin dashboard)
- `services/api` (Node.js + Express API)
- `services/ai` (Python + FastAPI AI service)
- `infra` (database, storage, deployment scaffolding)
- `docs` (architecture and platform docs)
